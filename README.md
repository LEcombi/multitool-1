# multitool v0.6
Simple Ethical Hacking multitool

## What can we do with it?
- Scan ports.
- ddos targets.
- bruteforce web directories.
- Check/Test scraped proxies.
- Use Shodan api to find vulnerabilities.


## Installation
- pip/3 install -r requirements.txt
